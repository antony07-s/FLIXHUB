# FlixHub

A Netflix-inspired movie streaming web application built using React.

## Features

- User Authentication
- Movie Search
- Trending Movies
- Popular Movies
- Watchlist
- Responsive Design
- Protected Routes

## Technologies

- React
- React Router
- Context API
- Redux
- Firebase
- TMDB API
- Tailwind CSS

## Installation

```bash
npm install
npm run dev
```
